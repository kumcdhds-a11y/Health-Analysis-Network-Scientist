이 페이지는 소개용 정적 페이지입니다. GitHub Pages는 Python·R·Streamlit을 실행하거나 로컬 SAS/XPT 데이터를 읽지 못합니다. 따라서 분석 도구 자체는 데이터가 있는 로컬/서버에서 실행합니다. 데이터는 기기를 떠나지 않습니다. 아래 샘플 리포트는 도구가 만들어내는 결과물의 예시입니다.

무엇을 하나

국민건강영양조사(KNHANES)와 미국 NHANES 원자료를 대상으로, 웹에서 노출·결과·보정·정의를 선택하면 복합표본 설계가중 분석이 결정론적으로 수행되고 아래 산출물이 나옵니다.

인터랙티브 분석

모든 변수를 노출/결과로 선택. 연속형 결과는 선형 β, 이분형은 로지스틱 OR로 자동 전환. Table 1 + 연관성 + 논문 Word.

Trajectory

결과를 정의하는 변수들의 연령별 설계가중 궤적.

역학 보고서

Table 1~6, crude/adjusted OR, 하위군+상호작용, PSM·IPTW·AIPW·G-computation·TMLE, Love plot·VIF·RCS·E-value.

트렌드 보고서

연도별 유병률·age-sex 표준화율·Projection·다구간 Joinpoint APC·음이항 예측.

ML 보고서

16개 모델(XGBoost·LightGBM·CatBoost·HistGB 등) 튜닝·전체지표·ROC/PR·threshold·calibration·SHAP·PDP. 정보누출 자동 차단.

조작적 정의 (논문 표준)

당뇨 ADA고혈압 JNC 대사증후군 harmonized NCEP 3/5이상지질혈증 다기준 지방간 NAFLD-LFS · HSI · CAP흡연·음주 never vs ever

보정변수는 노출·결과·정의성분·체격군집을 자동 제외해 과보정을 방지합니다. 통계는 R survey 설계가중이며, LLM은 보고서 문장만 작성하고 수치를 재계산하지 않습니다.

로컬 실행

1. 설치

git clone https://github.com/<계정>/<저장소>.git
cd <저장소>
pip install -r requirements.txt
sudo apt-get install -y r-base r-cran-survey r-cran-jsonlite r-cran-rms r-cran-mass r-cran-sandwich

2. 데이터 배치 (기기 밖으로 나가지 않음)

./data/KNHANES/hn08_all.sas7bdat , hn08_dxa.sas7bdat , ...
./data/NHANES/demo_j , alq_j , bmx_j ...   (.sas7bdat 또는 .XPT)

3. 실행

streamlit run factory_app.py
# 원격/병원망이면 SSH 터널 후 브라우저 http://localhost:8501
ssh -L 8501:localhost:8501 사용자@서버
